# @nebutra/ai-primitives

Status: **WIP**

Low-level primitives shared across the `@nebutra/ai-*` packages —
`scopedKey`, `sha256`, `cosineSimilarity`, `clamp`, `estimateTokens`.
Pure, dependency-free utilities consumed by `code-index`,
`knowledge-graph`, `knowledge-rag`, and `agent-runtime`.

Interfaces are still settling; expect breaking changes until 1.0.
