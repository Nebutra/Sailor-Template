# @nebutra/knowledge-graph

Status: **WIP**

Knowledge-graph capability for the Nebutra ecosystem. This package
provides graph-shaped knowledge primitives that complement
`@nebutra/knowledge-base` (vector retrieval) and `@nebutra/code-index`
(provider-agnostic indexing).

Not yet wired into any production app — interfaces are still settling
and the production graph-store adapter is sidecar-gated. Expect
breaking changes.
