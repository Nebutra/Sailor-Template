# @nebutra/founder-cemetery

Ethical close-company flow, postmortem analysis, lesson extraction, and memorial
publication artifacts.

## Boundary

This package does not delete companies. It starts with pause, then lets the
founder choose archive or memorial publication with consent and a cooling-off
period.

## Commands

```bash
pnpm founder-cemetery:doctor
pnpm founder-cemetery:quickstart
pnpm founder-cemetery:debug
```

## Current Closure

- Pause-first closing flow.
- Deterministic death-cause inference over timeline summaries.
- Evidence-backed lesson records.
- Memorial artifacts with explicit consent.
- Cooling-off metadata for publication.
