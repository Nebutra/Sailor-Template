# @nebutra/outreach-engine

Status: **WIP**

Compliance-first outreach Play product.

The package creates auditable campaign drafts. It does not own lead databases,
email sender credentials, or CRM write adapters.

```bash
pnpm outreach-engine:doctor
pnpm outreach-engine:quickstart "decision makers at mid-size D2C companies"
pnpm outreach-engine:debug
```
