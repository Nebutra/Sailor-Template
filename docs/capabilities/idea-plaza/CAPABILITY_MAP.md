# idea-plaza Capability Map

## Boundary

`idea-plaza` is an ecosystem-product package. It creates explicit public idea
snapshots and fork lineage records.

## Owns

- Opt-in publish contracts.
- Sensitive-field scan before publish.
- Publication level metadata.
- Fork attribution records.
- Cemetery warning metadata on idea cards.

## Does Not Own

- Private founder data.
- Global registry transport.
- Human moderation queues.
- Cross-tenant data sharing.

## Current Closure

- `pnpm idea-plaza:doctor`
- `pnpm idea-plaza:quickstart`
- `pnpm idea-plaza:debug`
