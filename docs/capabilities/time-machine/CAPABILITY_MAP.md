# time-machine Capability Map

## Boundary

`time-machine` is an ecosystem-product package. It turns event-log history into
founder-facing timeline, branch, compare, and annotation records.

## Owns

- Timeline view data contracts.
- Branch request product metadata.
- Compare narratives over affected assets.
- Star annotations.
- Rollback dry-run presentation.

## Does Not Own

- Event truth.
- Content truth.
- Rollback apply.
- Media diff workers.
- Agent runtime state.

## Current Closure

- `pnpm time-machine:doctor`
- `pnpm time-machine:quickstart`
- `pnpm time-machine:debug`
