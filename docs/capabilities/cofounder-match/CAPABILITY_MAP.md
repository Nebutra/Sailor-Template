# cofounder-match Capability Map

## Boundary

`cofounder-match` is an ecosystem-product package. It builds activity-derived
profiles, scores complementarity, and opens mutual-consent intro artifacts.

## Owns

- 30-day activity gate.
- Behavior-derived skill vectors.
- Explainable match score.
- Mutual interest checks.
- Intro handoff artifact.

## Does Not Own

- Identity verification providers.
- Chat transport.
- Social graph truth.
- Reputation enforcement.

## Current Closure

- `pnpm cofounder-match:doctor`
- `pnpm cofounder-match:quickstart`
- `pnpm cofounder-match:debug`
