# outreach-engine Capability Map

## Boundary

`outreach-engine` is a play-product package. It creates compliance-gated
campaign drafts for founder-led outbound work.

## Owns

- ICP normalization.
- Local campaign draft structure.
- Email sequence templates.
- Compliance checks before any send adapter is allowed.

## Does Not Own

- Lead databases.
- Email sender credentials.
- CRM write adapters.
- Browser automation.

## Current Closure

- `pnpm outreach-engine:doctor`
- `pnpm outreach-engine:quickstart`
- `pnpm outreach-engine:debug`
